Applied Mechanics — All key formulas

1. Stress Analysis 2D

2D Rotation — Normal stress (x′)
$$\sigma_{x'x'} = \frac{\sigma_{xx}+\sigma_{yy}}{2} + \frac{\sigma_{xx}-\sigma_{yy}}{2}\cos 2\theta + \sigma_{xy}\sin 2\theta$$
2D Rotation — Normal stress (y′)
$$\sigma_{y'y'} = \frac{\sigma_{xx}+\sigma_{yy}}{2} - \frac{\sigma_{xx}-\sigma_{yy}}{2}\cos 2\theta - \sigma_{xy}\sin 2\theta$$
2D Rotation — Shear stress
$$\sigma_{x'y'} = -\frac{\sigma_{xx}-\sigma_{yy}}{2}\sin 2\theta + \sigma_{xy}\cos 2\theta$$
Principal stresses (2D)
$$\sigma_{I,II} = \frac{\sigma_{xx}+\sigma_{yy}}{2} \pm \sqrt{\left(\frac{\sigma_{xx}-\sigma_{yy}}{2}\right)^2 + \sigma_{xy}^2}$$
Principal angle
$$\tan 2\theta_p = \frac{2\sigma_{xy}}{\sigma_{xx}-\sigma_{yy}}$$
Maximum shear stress
$$\tau_{max} = \sqrt{\left(\frac{\sigma_{xx}-\sigma_{yy}}{2}\right)^2 + \sigma_{xy}^2} = \frac{\sigma_I - \sigma_{II}}{2}$$
Max shear direction
$$\tan 2\theta_{max} = -\frac{\sigma_{xx}-\sigma_{yy}}{2\sigma_{xy}}, \quad \theta_{max} = \theta_p \pm 45°$$
Cauchy’s formula
$$\mathbf{t}^n = \boldsymbol{\sigma}\mathbf{n}$$
3D transformation
$$\boldsymbol{\sigma}' = \mathbf{A}\boldsymbol{\sigma}\mathbf{A}^T$$
3D characteristic equation
$$\sigma_P^3 - I_1\sigma_P^2 + I_2\sigma_P - I_3 = 0$$
Mohr’s circle
Center $C = \left(\dfrac{\sigma_{xx}+\sigma_{yy}}{2},\, 0\right)$, Radius $R = \tau_{max}$. Sign convention: positive shear (clockwise) plotted negative.

2. Inhomogeneous Cross-Sections

Strain field
$$\varepsilon(y,z) = \varepsilon + \kappa_y z + \kappa_z y$$
Hooke’s law (local)
$$\sigma(y,z) = E(y,z)\,\varepsilon(y,z)$$
Axial stiffness
$$EA = \sum_i E_i A_i$$
NC position
$$z'_{NC} = \frac{\sum_i E_i A_i z'_i}{\sum_i E_i A_i}, \qquad y'_{NC} = \frac{\sum_i E_i A_i y'_i}{\sum_i E_i A_i}$$
Weighted inertia
$$EI_{yy} = \sum_i E_i I_i \quad \text{(parallel axis theorem per region)}$$
Axial strain (pure N at NC)
$$\varepsilon = N/EA$$
Axial stress
$$\sigma_i = E_i\,\varepsilon$$
Curvature (pure bending)
$$\kappa_y = M_y / EI_{yy}$$
Bending stress
$$\sigma_i(z) = E_i\,\kappa_y\,z$$
Shear stress
$$\tau(z) = -\frac{V \cdot ES_y^a}{b(z) \cdot EI_{yy}}, \qquad ES_y^a = \sum_k E_k A_k^a z_k$$
Multi-material shear ratio
$$\tau_k/\tau_l = E_k/E_l \quad \text{(when } b_k = b_l\text{)}$$
Deflection (simply supported, central load)
$$w = \frac{FL^3}{48\,EI_{yy}}$$

3. Buckling

Euler’s load
$$P_E = \frac{\pi^2 EI}{l^2}$$
General critical load
$$P_C = \frac{\pi^2 EI}{l_k^2}, \qquad l_k = k_e \cdot l$$
Effective length table
Cantilever $k_e=2$  |  Pin-pin $k_e=1$  |  Pin-fixed $k_e=0.7$  |  Fixed-fixed $k_e=0.5$
Slenderness
$$\lambda = l/r_i, \qquad r_i^2 = I/A$$
Relative slenderness
$$\lambda_r = \frac{\lambda}{\pi}\sqrt{\frac{\sigma_y}{E}}$$
Buckling stress
$$\sigma_E = E\left(\frac{\pi}{\lambda}\right)^2, \qquad \frac{\sigma_E}{\sigma_y} = \frac{1}{\lambda_r^2}$$
Eccentric tip deflection
$$\delta = e\frac{1-\cos\alpha l}{\cos\alpha l}, \qquad \alpha^2 = P/EI$$
Amplification factor
$$\frac{1}{\cos(\alpha l)}$$
Max stress (combined)
$$\sigma_{max} = \frac{P}{A} + \frac{Pe\,y_{max}}{I\cos(\alpha l)}$$
Design interaction
$$\frac{P}{A\gamma\sigma_u} + \frac{My_{max}}{I\sigma_u} \le 1$$

4. Plastic Analysis

Yield moment (rectangular)
$$M_y = \frac{1}{6}\sigma_y bh^2$$
Plastic moment (rectangular)
$$M_p = \frac{1}{4}\sigma_y bh^2$$
Shape factor (rectangular)
$$\alpha = M_p/M_y = 1.5$$
Yield moment (general)
$$M_y = \sigma_y I/\tilde{z}$$
Plastic moment (general)
$$M_p = \frac{1}{2}\sigma_y A_{tot}(\bar{z}_1 + \bar{z}_2)$$
Plastic NA
Divides section into $A_1 = A_2 = A_{tot}/2$
Full plastic normal force
$$N_p = \sigma_y A$$
Full plastic shear
$$V_p = \sigma_y bh/\sqrt{3}$$
M-N interaction
$$M/M_p + (N/N_p)^2 = 1$$
M-V interaction
$$M/M_p + (V/V_p)^2 = 1$$
Moment-curvature
$$M/M_y = 1.5 - 0.5(\kappa_y/\kappa)^2$$
Virtual work
$$W_e = W_i$$
Standard results
Fixed-fixed UDL: $q_c = 16M_p/l^2$  |  Propped cantilever midspan: $P_c = 6M_p/l$
